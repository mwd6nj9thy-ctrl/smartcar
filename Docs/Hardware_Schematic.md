## 时序图

### PWM输出时序 (电机速度控制)

```
PA6/PA7 PWM信号:
┌─ 占空比 ─┐
│          │
┌──┐       ┌──┐       ┌──┐
│  │       │  │       │  │
└──┴───────┴──┴───────┴──┴─────

周期: 100μs (10kHz)

速度级别:
  占空比 25%  → 低速
  占空比 50%  → 中速
  占空比 95%  → 高速
```

### USART2通信时序 (蓝牙)

```
波特率: 9600 bps
格式: 8N1 (8位数据, 无奇偶校验, 1位停止位)

时间轴: 1 bit = 104μs

┌─────────────────────────────────┐
│ Start │ D0 D1 D2 D3 D4 D5 D6 D7│ Stop │
│  0    │  ?  ?  ?  ?  ?  ?  ?  ? │  1   │
└─────────────────────────────────┘

例: 传输字符 'F' (0x46)
   二进制: 01000110
   时序: Start-0-1-1-0-0-0-1-0-Stop
```

### LCD1602控制时序

```
4位模式时序:

      ┌─────┐         ┌─────┐
  RS  │     │         │     │
      └─────┘         └─────┘
      (0=命令 1=数据)

      ┌─────┐
  RW  │     │  (始终=0,写入模式)
      └─────┘

          ┌────┐       ┌────┐
  E   ────┘    └───────┘    └──
            (使能脉冲)

  D7-D4 [高4位]  [低4位]
```

## 电机驱动连接详解

```
         ┌──────────────────────┐
   PA0 ──┤ IN1                  │
         │                      │
   PA1 ──┤ IN2    L298N Motor   │
         │    Dual H-Bridge     │
   PA8 ──┤ IN3                  │
         │                      │
  PB15 ──┤ IN4                  │
         │                      │
   PA6 ──┤ ENA       ┌────────┐ │
  (PWM)  │      →   ┌─ OUT1/2 ├─┼─▶ [M1- M1+] (左电机)
         │          │ │      │ │
   PA7 ──┤ ENB       └─ Out3/4 ├─┼─▶ [M2- M2+] (右电机)
  (PWM)  │           └────────┘ │
         │                      │
   GND ──┤ GND                  │
  +12V ──┤ +VCC                 │
         └──────────────────────┘

电机驱动真值表:

  IN1  IN2  | OUT1  OUT2 | 功能
  ──────────┼────────────┼─────────
   0    0   |  0     0   | 停止
   1    0   |  1     0   | 正向
   0    1   |  0     1   | 反向
   1    1   | (反向保护)  | 禁止

ENA/ENB: PWM速度控制 (0-100%占空比)
```

## 循迹传感器连接

```
五路TCRT5000传感器:

  [S1] [S2] [S3] [S4] [S5]
   │    │    │    │    │
   PB0  PB1  PB2  PB3  PB4
   │    │    │    │    │
   └────┴────┴────┴────┘
        └──── GND ────┘

逻辑:
  传感器输出 = 低电平 → 检测到黑线
  传感器输出 = 高电平 → 未检测到黑线

传感器排列 (俯视图):

   S1  S2  S3  S4  S5
   ●   ●   ●   ●   ●   (向下)
   
  ←────── 行进方向 ──────→

循迹算法:
  全检测到: S1,S2,S3,S4,S5 全低 → 直行
  中心检测到: S3 低 → 直行
  左偏: S2或S1 低 → 左转
  右偏: S4或S5 低 → 右转
```

## 超声波传感器时序

```
HC-SR04 工作流程:

1. 触发信号 (PB5):
   ┌──────┐
   │ 10μs │
   ├──────┤
   │ PWM  │ ← 输出10μs脉冲
   └──────┘

2. 返回信号 (PB6):
   ┌────────────────────────────────┐
   │         高电平时长              │
   ├────────────────────────────────┤
   │ t = (时长 / 2) / 340 m/s * 10^6│
   │ 距离(cm) = t / 58              │
   └────────────────────────────────┘

   例: 高电平200μs
       距离 = 200 / 58 ≈ 3.4 cm

3. 测量周期: ≥60ms
```

## 液晶屏接口

```
LCD1602 4位模式:

         ┌────────────────────────┐
    E ───┤1  ENABLE               │
         │  (使能信号,下降沿锁存) │
         │                        │
   RS ───┤2  REGISTER SELECT      │
         │  (0=命令 1=数据)       │
         │                        │
   RW ───┤3  READ/WRITE           │
         │  (0=写入 1=读取)       │
         │                        │
   D7 ───┤4-7  DATA LINES (D4-D7)│
   D6 ───┤                        │
   D5 ───┤  (仅用高4位)           │
   D4 ───┤                        │
         │                        │
   +5V ──┤8  VDD                  │
   GND ──┤9  VSS                  │
         │                        │
   POT ──┤10 VO (对比度)         │
         │  (10K电位器)           │
         │                        │
   +5V ──┤11 A (背光+)            │
   GND ──┤12 K (背光-)            │
         └────────────────────────┘

4位模式时序:

命令/数据发送:
1. 设置 RS (0=命令,1=数据)
2. 设置 D7:D4 (高4位数据)
3. E脉冲 (下降沿锁存)
4. 设置 D7:D4 (低4位数据)
5. E脉冲 (下降沿锁存)
```
